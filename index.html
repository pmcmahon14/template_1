Road Bound

Personalize your strech of ribboned highway!

Technologies Used

jQuery, JavaScript, PHP, AJAX, Google Maps API, Google Places API, Weather Underground API, Energy Information Administration API, Eventful API, HTML5/CSS3

Description

This app was developed in five weeks with three other developers. It allows the user to create an account, plot a trip, create lists for a trip, and find events and stops along the route. The user can also set a budget for a trip (including fuel costs) and keep track of weather conditions along the way.

Find Food Fast

Hungry and on the road?

Technologies Used

jQuery, JavaScript, Google Places API, Google Directions API, YouTube API, AJAX, HTML5/CSS3

Description

This mobile-only app was developed in two days with three other developers. It allows the user to find a place to eat from nine popular chains based on the direction the user is travelling. It requires minimal interaction and locates stops along your way.

Tic-Tac-Toe

More than X's and O's!

Technologies Used

jQuery, JavaScript, Bootstrap, Leap Motion Control, HTML5/CSS3

Description

Created in two day with two other developers, this puts a modern twist on Tic-Tac-Toe. It features visual and sound effects, two different grid sizes, a timer for each turn, a responsive board, and an option for virtual reality with a Leap Motion console.

Guessing Game

Boom, confetti!

Technologies Used

jQuery, JavaScript, HTML5/CSS3

Description

This is a festive take on the guessing game. Through using canvas and animation, I learned how to make the board responsive as game conditions changed and to create a celebration upon a win. It features a range display, a changing background color based on how many guesses are left, and a canvas confetti display and sound effect when the player guesses the correct number.
